			project "EVIL GENE"(いびるじーん)

概要
	真上見下ろし型ラウンド式ゾンビシューター。

プレイ人数
	1〜4（予定）
		本当にマルチを実装するなら人数ごとに敵を強化するなり増やすなりする

キャラコン
コントローラーでの操作
	左スティック : 移動
		押し込み : スプリント
	右スティック : キャラの向き及び照準
		押し込み : 緊急格闘攻撃
	LBボタン : 射撃
	RBボタン : 投擲武器使用
		長押しで投擲距離延長
	LTボタン : メディカルキット1つを消費
	RTボタン :マップ全体表示
	Yボタン : 主要武器変更
		主要武器は2つ持てる
		長押しでナイフ
	Xボタン : 武器の再装填
	Bボタン : 各種アクション、メニューにおける決定等。
	Aボタン : 柵の乗り越え、メニューにおけるキャンセル等。
	十字ボタン : 投擲武器変更
		↑ : フラググレネード(デフォルト)
		→ : フラッシュバン
		↓ : モロトフカクテル
		← : トマホークアックス

移動速度に関する仕様
	後ろ歩きに近いほど遅くなる。なんてことはない。
	射撃中は遅くなる

スプリントの仕様
	スプリントは向いている方向に自動的に進む
	スプリント解除後は少しの間武器が使えない
	スプリント中はスタミナが減る。

	主要武器のステータス及び必要変数の見当
		dmg : 弾が敵に与えるダメージ。Sarvoに限ってペレット1個が敵に与えるダメージ。
		agi : 射撃中移動速度。要はLBボタン押下中の移動速度。サブ1で表記
		spd : 弾の拡散の最大角。
		fir : 発射速。次の発射まで何フレーム必要かの値。
		prt : 現状Sarvo以外に存在しないステータス。
		rng : 射程距離。何フレーム飛ぶか
		pic : 貫通力。何体敵を貫通するかの指数。Hawk等が高くSarvo、Mamba等が低い。
		nob : 装弾数。
		fit : 射撃タイプ。現在3つ 
			Full(LBボタンを押下している間射撃。 例:M68,Mamba)
			Semi(LBボタンを押下したとき射撃。 例:G762,Tr9mm)
			Bolt(LBボタンを押下のち離したとき射撃  例:Hawk)
		crp : クリティカル発生率。ライフル系が高い。
		crd : クリティカル時ダメージ倍率。ライフル系以外が高い。Hawkはその限りではない
		rlt : リロード所要時間。フレーム数。
		cst : コスト。取得に必要なポイント
			アップグレード要素あり
		
		ポイントの呼称
			「クレジット」と呼称する。敵を攻撃したり倒したりすると手に入る。
			武器の解放、強化、補充、メディカルキットの補充、パークの購入、エリア解放等ができる。略称はcr
			
		クレジットの入手
			敵を攻撃したり倒した際に入手できる
		
		武器の取得
			マップの所々に箱があり、そこでクレジットを使って解放、強化、変更ができる。
			
		武器強化
			クレジットを使って武器のスペックを底上げできる
				・sight・・・照準器。クリティカル率、命中精度を上げる。
				・magazin・・・弾倉。装弾数、リロード速を上げる。
				・grip・・・グリップ。射撃中移動速度を上げる。
				・ammo・・・弾薬。主にダメージを、時々貫通力を上げる。sarvoはダメージ、ペレット数。
				・barrel・・・銃身。クリティカルダメージ、連射速度を上げる。
			
		弾薬の補給
			これも箱で行える。武器コスト半分以下のポイントで弾薬を全快する。
			低確率で敵が落とすこともある。

	投擲 : 	それぞれ3個ずつ持てる。デフォルトではフラググレネードが選択されている。
			箱で補給できるほか、敵が落とすこともある。
			
		フラググレネード : 中程度の範囲にダメージを与える。
		フラッシュバン : 広範囲の敵を行動不能にする。
		モロトフカクテル : 広範囲に持続ダメージを与える炎を発生させる。
		トマホークアックス : 敵単体に大ダメージを与える。
	十字ボタンで選択したものをRBボタンで投げる感じ。

メディカルキットの効果
	体力を即最大まで回復するのに加えて最大体力が一時的に増える。
	最大3個もてる

メディカルキットの補給
	箱で行えるほか、敵が低確率で落とすこともある。
	補給コストは1000
	
	パークについて
		マップ各所に散らばるダウンしない限り永久的に強力なバフを与えてくれるアイテム。
		パークは最大で3つまでしか取得できない。それ以上購入した場合、古いパークから除去されるので注意。
		PretendDocを使用したり、PretendDocを持っていない状態で他者から蘇生されたりすると全パークが剥がれる。
		パークは何度失っても購入できるが、コストが高いので、ご利用は計画的に。
	
	パーク
		・PretendDoc・・・購入後1回限りで自己蘇生できる。1人でやるならほぼ必須。必須すぎてアミノ酸。
		  2人以上のプレイに限り、パークの保険としても働く、その場合でも当パークだけは消える。必要cr 2000
		・Juggernaut・・・武器全般の与ダメージが1.5倍になる。必要cr 2000
		・DoreadNought・・・受けるダメージが1/2になる。必要cr 2000
		・AloneWar・・・投擲武器をそれぞれ最大5つ持てるようになるのに加え、最大弾薬数が1.5倍される。 必要cr 2000
		・OliveCrown・・・移動速度全般が1.25倍になる。スタミナ消費が半分になる。必要cr 2000
	
敵に関する事
	敵の種類
		ノービス : 	親の顔より見る羽目になるゾンビ。
					1体1体はとろいので脅威ではないが、数が多いので囲まれると死ぬ
					。
		ハウンド : 	犬ゾンビ。すばやい、つよい、で完全にノービスの上位互換。
					数は少ない。15ウェーブごとに大量発生する。ひるみやすい。
					
		イレギュラー : 	10数ウェーブか一回に1体ないし数体出現する中ボス系ゾンビ。
						多種多様である、実に。
						
	イレギュラーの種類
		ブルート : 	こっちを視界にとらえるなりタックルしてくる人。
					タックルからの追撃もセットにする。警察官っぽい。
		
		ドク : 	放射能をばらまきながら近寄ってくる防護服を着こんだゾンビ。
				固いうえ近くにいるだけでダメージを受けるため接近戦は禁物。
				
		タンク : 	固いうえにゲロみたいな飛び道具を使ってくるドド●アっぽい敵。
					近寄りすぎるとそこそこ痛い殴り攻撃をしてくる。
					
		ヘルハウンド : 	ハウンドの強化型。見た目は通常と変わらないけど、こっちのほうが大柄。
						スプリントしないと逃げられないほど速いけど、通常と同様ひるみやすい。
						
		ネメシス : 	強い、固い、飛び道具、の三拍子がそろう。ただしノービスレベルにとろい。
					飛び道具はタンクレベルのダメージはないが、弾速が段チ。
